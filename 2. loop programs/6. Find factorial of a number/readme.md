# Program to find factorial of a number

## Author

## Discription
This code wlii calculate the factorial of a number by user input a number.
The factorial of a number is the product of all positive integers less than or equal to that number. For example, 5! = 5 × 4 × 3 × 2 × 1 = 120

# How it works
Uses an iterative loop to multiply numbers from 1 to n.
entered number must be greater than or equal to 1.
