# Print even number from 1 to 100

## Author
Prakash Singh

## Discription
This is a C program to print even numbers from 1 to 100.
The for loop starts with i set to 2 and increments i by 2 in each iteration until i is greater than 100. 
The printf function is used to print each number on a new line.
The printed numbers will be all even numbers.
