# Multiplication table of a number

## Author
Prakash Singh

## Discription
This is a programto print multiplcation table by giving number as input by user.
for loop iterates from 1 to 10, printing the product of the number and the current iteration value.

## How it works
Input Validation: Uses scanf return value to ensure the user enters a valid integer.
Loop: A for loop runs from 1 to 10 to generate the multiplication table.
Negative Numbers: Works for both positive and negative integers.
Output Formatting: Displays results in a clean, readable format.
