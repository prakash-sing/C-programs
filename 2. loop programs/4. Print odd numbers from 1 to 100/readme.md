# Print odd numbers from 1 to 100

## Author
Prakash Singh

## Discription
This is a simple logic building C program to print odd numbers from 1 to 100 using for loop.
The for loop starts with i set to 1 and increments i by 2 in each iteration until i is greater than 100. 
The printf function is used to print each number on a new line.
Printed numbers will odd numbers.
