# Check the Armstrong number

## Author
Prakash Singh

## Discription
This is a code for check whether a number is armstrong number or not using while loop.
An Armstrong number (or narcissistic number) is a number that is equal to the sum of its digits raised to the power of the number of digits. For example, 153 is an Armstrong number because: 1³ + 5³ + 3³ = 153.

## How It Works
Input: The user enters a number.
Digit Count: The program calculates the total number of digits in the input.
Sum Calculation: Each digit is raised to the power of the total digit count and summed.
Comparison: The sum is compared with the original number to determine if it’s an Armstrong number.
