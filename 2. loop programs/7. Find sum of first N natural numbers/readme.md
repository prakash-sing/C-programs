# Sum of first N natural numbers

## Author
Prakash Singh

## Discription
This is C program to calculate and print the sum of the first n natural numbers.
Using for loop it iterates from 1 to entered number by user and add all of them.
# How it works
It take Input from User Prompt the user to enter a positive integer n.
Initialize Variables Set sum = 0 to store the cumulative total.
Use a Loop to Add Numbers Run a for loop from 1 to n, adding each number to sum.
Display the Result Print the final value of sum.
