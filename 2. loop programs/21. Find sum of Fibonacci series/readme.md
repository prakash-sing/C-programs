# Sum of Fibonacci series

## Author
Prakash Singh

## Discription
This is a code to print the sum of the Fibonacci series up to n terms using a .
It includes input validation and handles edge cases like n <= 0.

## How it works
Initialization: first = 0, second = 1 are the first two Fibonacci numbers.
Loop: For each term, calculate the next Fibonacci number.
Add it to sum.
Output: Prints the series and the total sum
