# Square of a number by return type function

## Author
Prakash Singh

## Discription
This program prompts the user to enter a number, calculates the square using the square function, and displays the result.
The square function is a simple return type function that takes an integer as an argument and returns the square of that number. 
This program includes - 
- The main function calls the square function and prints the result. 
- Uses simple multiplication instead of complex math functions.
- Demonstrates return-type functions clearly.
- Includes basic input validation.

## How it works
Function square
- Takes an integer as input.
- Returns the square of that integer.
- Main function
Reads an integer from the user.
- Validates the input.
- Calls square() and prints the result.
