# Calculate factorial value of an integer using a function

## Author
Prakash Singh

## Description
this is a c program to calculate factorial of a number using function.
It prompts user to enter an integer a user input.
Entered number must be a positive number.
## How it works
The program prompts the user to enter an integer and stores it in variable n.
A function fact() is declared to calculate factorial.
A for loop iterates from 1 to n, multiplying each value with the variable factorial.
Then the calculated factorial is displayed.
