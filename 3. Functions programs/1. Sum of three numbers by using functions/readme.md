# Calculate sum of three numbers by user defind functions

## Author 
Prakash Singh

## Description
This is a complete runnable C program to calculate the sum of three numbers using a function.
It is user input program and asks user to enter three numbers.

## How it works
calsum() function: Takes three integers and returns their sum.
readInt function: Prompts the user and validates integer input.
Main program: Reads three integers from the user.
Calls calsum() to compute the sum.
And now finally prints the result.
