# Even odd checking using switch

## Author
Prakash Singh

## Description
A simple c program to checkin of even odd using switch operator.
## How it works
The program takes an integer input from the user.
It calculates the remainder when the number is divided by 2 (num % 2).
- If the remainder is 0, the number is even.
- If the remainder is 1, the number is odd.
The switch statement evaluates the remainder and executes the corresponding case.
