# Menu driven program of factorial, prime check and odd-even check

## Author
Prakash Singh

## Description
This meenu driven program has the following options:
- Factorial of a number
- Prime or not
- Odd or even
- Exit
Once a menu item is selectedd the appropriate action should be taken at once this action finished, the menu reappear.
Unless the user select the 'Exit' option the program should continue to work.
