# Print multiplication table using switch

## Author
Prakash Singh

## Description
Here’s a complete, runnable, and robust C program to print a multiplication table using a switch-case statement.
It includes input validation and handles edge cases like negative numbers and zero.

## How it works
Menu Display :– The program shows two options:
- 1 → Print multiplication table
- 2 → Exit
Switch-Case :– Executes code based on the user’s choice.
Input Validation – Ensures only valid integers are processed.
Edge Cases :– If the number is 0, it prints that all results are 0.
Works for negative numbers as well.
