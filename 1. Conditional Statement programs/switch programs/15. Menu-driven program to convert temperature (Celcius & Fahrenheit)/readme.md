# Menu-driven C program to convert temperatures

## Author
Prakash Singh

## Description
this is menu-driven C program using switch to convert temperature between Celsius and Fahrenheit.

## How it works
Menu Display – Shows two options:
1 → Celsius to Fahrenheit
2 → Fahrenheit to Celsius
Switch Statement – Executes the correct formula based on the user’s choice.
Formulas Used:
- Fahrenheit → Celsius: F=(C× 5/9)+32
- Celsius → Fahrenheit: C=(F−32)× 9/5
Input Validation – Checks if the user entered valid numbers.
