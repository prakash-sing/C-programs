# Menu-driven program for matrix operation as addition and subtraction

## Author
Prakash Singh

## Description
This is a simple menu-driven C program using a switch statement to perform matrix addition and subtraction.

## How it works
The program first takes the dimensions and elements of two matrices from the user.
It displays a menu with options for matrix addition, subtraction, or exiting the program.
Based on the user's choice, the program performs the selected operation using nested loops and displays the result.
The program continues until the user chooses to exit.
