# Check whether a character is an alphabet or a digit or a special symbol

## Author
Prakash Singh

## Description
This is a C-program to check entered chaaracter is an alphabet, a digit or a special symbol.
This program uses a switch case creatively by leveraging conditions inside the cases.

## How it works
Alphabet Check: The program checks if the character falls within the ASCII range of uppercase ('A' to 'Z') or lowercase ('a' to 'z') alphabets.
Digit Check: It checks if the character is between '0' and '9'.
Special Symbol: If neither of the above conditions is true, the character is classified as a special symbol(in defult value).
