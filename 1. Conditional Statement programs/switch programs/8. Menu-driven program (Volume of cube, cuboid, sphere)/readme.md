# Menu driven program to calculate volumr of cube, cuboid or sphere

## Author
Prakash Singh

## Description
Menu Display: The program displays a menu with three options for calculating the volume of a cube, cuboid, or sphere.
Switch Statement: Based on the user's choice, the corresponding case is executed.
Formulas Used:
- Cube : Volume = side*side*side.
- Cuboid : Volume =length×breadth×height.
- Sphere : Volume = 3/4*πr*r*r.
Error Handling: If the user enters an invalid choice, an error message is displayed.
This program is simple, user-friendly, and demonstrates the use of switch effectively.
