# Check whether a triangle is valid or not

## Author
Prakash Singh

## Description
User input program. 
As per triangle rule the triangle is valid if the sum of two sides is greater than the largest of the three sides.

## How it works
Prompts user to input three integers as the three sides of the triangle.
In the entered three sides of the triangle.
The triangle is valid if the sum of two sides is greater than the largest of the three sides.
Then the answer is printed accordingly.
