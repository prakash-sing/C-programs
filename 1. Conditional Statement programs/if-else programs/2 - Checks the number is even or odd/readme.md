# Check whether a number is even or odd

## Author
Prakash Singh

## Discription
This program is for determining the number is even or odd by using modulus(%) operator to check the number is divisible by 2 or not.
the program gives an output whether the number is even or odd.

## Features
- User enter an integer.
- Uses the modulus operator to check divisibility by 2.
- Prints the output that the number is even or odd with input.
- Simple and easy to understandable code for beginners.

## How it works
- The program firstly request user to enter a number.
- 'scanf' reads the entered number and stores in the variable 'num'.
- The modulus operator '%' calculates the reminder after divided by 2.
- If reminder gives 0 then the number is even, otherwise it is odd.
- The answer is printed accordingly.
