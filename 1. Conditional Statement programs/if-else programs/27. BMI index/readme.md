# checkiung the BMI of a person

## Author
Prakash Singh

## Description
This is a C-program for checking a person's BMI index using first Weights (in kg) and second Square of height (in meters).
Prompts user to enter the two values weight and height.
By entering the weight and height it calculate the BMI by entered values. 

BMI category        BMI
- Starvation          <15
- Anorexic            15.1 to 17.5
- Underweight         17.6  to  18.5
- Ideal               18.6  to  24.9
- Overweight          25  to  25.9
- Obese               30  to  30.9
- Morbidly Obese      >=40
