# Find the youngest among three people

## Author
Prakash singh

## Description
The program takes age of three people in integer as inputs from the user and determines which one is youngest among them.
It demonstrates the use of basic input/output functions and conditional statement(if-else) and condition into it in C-programming.

## Features
- Prompts the user to enter three integers.
- Compares the three numbers to find the smallest.
- Displays smallest number.
- Simple and effective logic suitable for beginners to learn C.

## How it works
- The program asks the user to enter three numbers as input.
- It reads the input using 'scanf'.
- Then it compares the three numbers in conditional statement:
 - First it compares the first two numbers.
 - Then compares the smaller one with the third number.
- Finally, it prints the smallest number as age.
