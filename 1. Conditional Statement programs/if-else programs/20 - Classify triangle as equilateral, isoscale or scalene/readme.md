# Classify triangle as equilateral, soscale or scalene

## Author
Prakash Singh

## Description
A user input C program to check triangle type on the basis of side.
beginner friendly with simple if else conditional program.

## How it works
Prompts user to enter three integers as input as side.
It contain simple if-else condition:
- Three sides equal = equilateral triangle.
- two sides equal = isoscale triangle.
- all sides different = scalane triangle.
