# Check whether a person is eligible to drive

## Author
Prakash Singh

## Description
This program is controlled by conditional statement .
Beginner friendly C-program to checks a person is eligible to vote or not.

## Features
Prompts to user to enter an integer input as age.
Checks by conditional statement.

## How it works
Checks the age of a person by user input age.
if-else statement classify the age is either less than 18 or 18 and greater than 18.
If age is less than 14 then appropriate output will shown not to drive.
Next if condition is if age is between 14 to 18 taen also an appropriate message shown to drive with condition.
Else it will print you are eligible to drive.
