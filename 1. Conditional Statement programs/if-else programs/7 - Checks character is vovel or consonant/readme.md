# Checking whether a character is vovel or consonant

## Author
Prakash Singh

## Description
This C-program checks first whether is the input character is an alphabet or not then it checks whether is it a vovel or a consonent.
This program displays an appropriate message of vovel or consonent if input is a vovel or a  consonant.

## Features
- Prompts the user to enter a single character.
- Evaluates if the entered character is an alphabet (A-Z or a-z).
- After its evaluation gives an alphabet then it evaluate whether is it a vovel or a consonant. 
- Prints a message indicating whether the character is an alphabet and after this is it a vovel or a consonant.
- easily understandable program for beginners.

## how it works 
- The program takes a character as input and checks if the ASCII value of the character falls in the range of uppercase(A-Z) or lowercase(a-z) alphabets.
- If yes, then it evaluate whether is it a vovel or  consonant.
- it prints that the character is an alphabet and after this it prints that it is a vovel or a consonant otherwise it print that it is not an alphabet.
