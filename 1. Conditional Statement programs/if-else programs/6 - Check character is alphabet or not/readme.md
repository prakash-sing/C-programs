# Checking whether a character is an alphabet or not

## Author
Prakash Singh

## Description
This C-program checks whether is the input character is an alphabet or not.
This program displays an appropriate message of an alphabet if the input is between `A-Z` or `a-z`.

## Features
- Prompts the user to enter a single character.
- Evaluates if the entered character is an alphabet (A-Z or a-z).
- Prints a message indicating whether the character is an alphabet or not.
- easily understandable program.

## how it works 
- The program takes a character as input and checks if the ASCII value of the character falls in the range of uppercase(A-Z) or lowercase(a-z) alphabets.
- If yes, it prints that the character is an alphabet otherwise it print that it is not an alphabet.
