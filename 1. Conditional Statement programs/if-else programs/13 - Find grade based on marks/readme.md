# Find grade based on the marks

## Author
Prakash Singh

## Discription
This is a C-program to determine grade based on marks.
The marks from 5 different subjects.
The program prompts the user to enter five integer and checks grade based on simple if-else conditional statement.

## Features
User enter five integer as marks in five subjects.
Checks grade by marks based on conditional statement.
show an appropriate result with grade based on input integers or marks.
