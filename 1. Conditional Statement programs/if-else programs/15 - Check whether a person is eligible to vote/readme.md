# Check a person is eligible to vote

## Author
Prakash Singh

## Description
I have write this program by Ternary operator `?`.
Beginner friendly C-program to checks a person is eligible to vote or not.

## Features
Condition ? doSomething if TRUE : doSomething if FALSE;

## How it works
Checks the age of a person by user input age.
Ternary operator classify by the age is either less than 18 or 18 and greater than 18.
If age is less than 18 then appropriate output will shown.
Else it will print you are eligible to vote.
