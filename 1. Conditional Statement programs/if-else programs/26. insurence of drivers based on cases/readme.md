# Insurence of drivers on a perticular conditions

## Author
Prakash singh

## Conditions/Cases
- If the driver is married.
- If the driver is unmarried, male and above 30 years of age.
- If the driver is unmarried, female and above 25 years of age.

## Description
Prompts user to user an integer and two character value as input.
- Age in numbers,
- sex if male enter 'M' and if female enter 'F',
- Marital status if unmarried enter 'U' and if married enter 'M'.
Checks by simple if else condition and relational operators(OR-AND).
